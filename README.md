# AutoML Recommendation System

## 🚀 Overview
This project recommends the best ML algorithm for a given dataset using:
- Meta-learning
- Similarity search (FAISS)

## ⚙️ Features
- Upload dataset (CSV)
- Extract meta-features
- Hybrid recommendation engine
- FastAPI backend

## 🧠 Tech Stack
- Python
- FastAPI
- Scikit-learn
- FAISS
